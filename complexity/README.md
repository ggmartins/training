## 1.1 O(1)

- Accessing values in a hash table by key

## 1.2 O(log n)

- Divide and conquer algorithms

## 1.3 O(n)

- Traversing a list

## 1.4 O(n log(n))

- Iterations that use divide and conquer

## 1.5 O(n^2)

- 2 Nested for loops

## 1.6 O(2^n)

- Naive recursive Fibonacci sequence calculation

## O(n!)

- Adding a nested loop for each loop
